Layer * Gorilla::LayerContainer::createLayer ( Ogre::uint  index = 0)

function. createLayer desc. Create a layer for drawing on to.

Index represents the z-order, 0 being the layer drawn first and 15 the layer drawn last. Layers drawn after another layer will appear to be top than the other.

note. Index must be between or equal to 0 and 15. Any other value will cause a very nasty crash.
